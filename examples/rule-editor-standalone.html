Rules Script Quick Reference

Structure:
  • RULE "name" - Start a rule
  • TYPE: INCLUSION | EXCLUSION | PRIORITY | BONUS
  • MANDATORY: YES | NO
  • WHEN condition
  • FAIL MESSAGE: "text"
Operators:
  • = != > >= < <=
  • BETWEEN x AND y
  • IN ("a", "b", "c")
  • CONTAINS STARTS WITH ENDS WITH
  • IS EMPTY IS NOT EMPTY
Logic:
  • AND - Both must be true
  • OR - Either can be true
  • NOT - Negate condition
  • ( ) - Group conditions
  • # - Comment line
